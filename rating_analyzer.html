舞レート解析ツール

chunithmにはプレイヤーのBEST Resultから到達可能Ratingを計算するツールがいくつか作られてますが、 maimaiにはありませんでした。

一方、内部Lvを計算するうえで、何らかの表計算ソフト（と計算マクロ）を作っておいて、 調査時にそれを使う方が楽なのは間違いないので、私もGoogle Spread Sheet （+ Google Apps Script）を使って作りました。

Google Apps ScriptはJavascriptに近いため、ほぼそのまま流用してそれらしきものを作り出すことに成功しました。

使い方

     javascript: (function() { var d=document; var i=location.hostname.indexOf('maimai-net.com'); if(i!=-1) { var url='https://sgimera.github.io/mai_RatingAnalyzer/maiRatingAnalyzer.js'; var e=d.createElement('script'); e.src=url; d.getElementsByTagName('head')[0].appendChild(e); } else { if(confirm('Do you open maimai.net?')) { window.open('https://maimai-net.com/'); } } })(); void(0);
