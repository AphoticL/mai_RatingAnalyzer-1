あならいざもどき（舞レート解析ツール）

現在たくさん（？）の方々のご協力により、Ratingの計算に関することや、内部Lv.が判明してますので、
chunithmには多数存在するレートアナライザなる物を作ってみました。

レートアナライザとは、現在のBEST SCOREのデータと内部Lv.を用いて、到達可能Ratingを算出するものです。

maimaiをプレーするうえでの一つの指標としてご利用ください。

3月9日より再公開しました。どうぞごひいきに。

使い方

事前準備として適当なブックマークを作っておき、maimai.netを開いた後でそのブックマークを選択する、というやり方となります。

事前準備

何でもよいので、ブックマークを一つ作りましょう。

  • タイトルは何でもよいです。ただ、アルファベットで始めておくと少し楽…かも。
  • URLの部分には下記を貼り付け。 
          javascript: (function() { var d=document; var i=location.hostname.indexOf('maimai-net.com');
      if(i!=-1) { var url='https://sgimera.github.io/mai_RatingAnalyzer/maiRatingAnalyzer.js';
      var e=d.createElement('script'); e.src=url; d.getElementsByTagName('head')[0].appendChild(e); }
      else { if(confirm('Do you open maimai.net?')) { window.open('https://maimai-net.com/'); } } })(); void(0);

操作

maimai.netを開いた後、先ほど作成したブックマークを選択します。

開くページは楽曲スコアのアイコンが見えるページであればどこでもよいです。
当然、TOPページでもよいですが、コレクションのページや、動画のページでも大丈夫です。

androidとchromeの組み合わせだと、ブックマークから選択しても反応してくれないようです。
（他の組み合わせでもあるかもしれません。）
その場合は、ブラウザのアドレスの所にブックマークのタイトルを入力して出てきた物をTAPすることで対応できます。

基本データと到達可能Rating

あならいざもどきにおける出力のメインです。各種値の説明は横に書いてます。なぜその説明なのかはサイト内文章でご確認ください。

ここに記載されている到達可能Ratingが、現在の実力で到達できる最高点の予想となります。
もちろん、あならいざもどき実行後にSCOREが伸びたり未プレー譜面をプレーしていけば、到達可能Ratingも伸びていきます。

予想である以上、間違っていることもあります。

最大値（あくまで予想ですが）については、”枠”の表のページの一番下にありますので、併せてご確認くださいませ。

全譜面データ（と呼んでいるもの）

ただの飾りです。（重要）

到達可能Ratingを算出する際の中間データを、バグ確認のために出力しているものです。あくまで参考データ。

よくある質問

Q : 到達可能Ratingが自分の最大Ratingより低い値になるのだけども。

A : 内部Lvのデータが出そろっていない場合、各Lvの最低値で計算してます。
そのため、予想値が実Ratingより下回る可能性はあり得ます。
あまりにずれがひどすぎたので、牛乳ver.で未検証の譜面は紫+ver.の値で代用してますが、正しいかどうかは不明です。
また、MASTERをあまりプレーしていなかったり、EXPERTのほうがレート値が高くなるような状況の場合、やはりズレます。


Q : 現在の最大Ratingは？

A : こちらも予想値となりますが、枠の表のページの一番下に、全譜面SSSを獲得できた時の予想最大値を書いてます。


Q : どうやったら到達可能Ratingに到達するの？

A : 単曲レート値が最大の曲（全譜面データの一番上…と言いたいところですが、今は適当なので…）で、
10回同じ成績（それより上が出せるならそれでもOK）を出し続けることで到達します。


Q : 全譜面データの30位（473位）と、BEST下限（History下限）の値が違うんだけど。

A : 全譜面データは飾りなので気にしないでください。一番上にある基本データが正しい値です。


Q : 今まで使えていたのに動かなくなった

A : 新しい機能を入れると、私のところで動くのに他の方が使えなくなることはたまにあります。
現在使用中のあならいざもどきのブックマークをコピー頂き、URLを下記に変更して実行頂いた結果を私に送っていただけますでしょうか。

      javascript: (function() { var d=document; var i=location.hostname.indexOf('maimai-net.com');
      if(i!=-1) { var url='https://sgimera.github.io/mai_RatingAnalyzer/maiRatingAnalyzer_debug1.js';
      var e=d.createElement('script'); e.src=url; d.getElementsByTagName('head')[0].appendChild(e); }
      else { if(confirm('Do you open maimai.net?')) { window.open('https://maimai-net.com/'); } } })(); void(0);
動かすと、
  1. タイトル
  2. 全譜面データ出力選択
  3. 名前とratingと段位 完了
  4. 楽曲データ 完了
  5. コレクション 完了
  6. 紫段位 完了
  7. Rank/Comp系 完了
  8. あとは表示だけ
と出てくるので、どこまで出てくるかを試してみてください。
