舞レート解析ツール

chunithmにはプレイヤーのBEST Resultから到達可能Ratingを計算するツールがいくつか作られてますが、maimaiにはありませんでした。

計算式と内部Lv.が見えてきたので、作ってみました。

maimaiをプレーするうえでの一つの指標としてご利用ください。

使い方

事前準備として適当なブックマークを作っておき、maimai.netを開いた後でそのブックマークを選択する、というやり方となります。

事前準備

何でもよいので、ブックマークを一つ作りましょう。

  • タイトルは何でもよいです。ただ、アルファベットで始めておくと少し楽…かも。
  • URLの部分には下記を貼り付け。 
          javascript: (function() { var d=document; var i=location.hostname.indexOf('maimai-net.com');
      if(i!=-1) { var url='https://sgimera.github.io/mai_RatingAnalyzer/maiRatingAnalyzer.js';
      var e=d.createElement('script'); e.src=url; d.getElementsByTagName('head')[0].appendChild(e); }
      else { if(confirm('Do you open maimai.net?')) { window.open('https://maimai-net.com/'); } } })(); void(0);

使い方

maimai.netを開いた後、先ほど作成したブックマークを選択します。

androidとchromeの組み合わせだと、ブックマークから選択しても反応してくれないようです。
（他の組み合わせでもあるかもしれません。）
その場合は、ブラウザのアドレスの所にブックマークのタイトルを入力して出てきた物をTAPすることで対応できます。
